# Copit

Copit は、macOS のメニューバーに常駐して使う軽量なクリップボード補助アプリです。  
`⌘ + Shift + C` で選択中のテキストを Copit の履歴へ保存し、`⌘ + Shift + V` で一覧を開いて素早く貼り付けできます。

## 一般ユーザー向け

### できること

- 選択中のテキストを Copit 専用の履歴に追加できます
- 履歴一覧から目的のテキストを選んで、そのまま現在のアプリへ貼り付けできます
- よく使う項目をお気に入り登録できます
- 通常履歴は最大 10 件まで保持され、お気に入りは保持数の制限を受けません
- メニューバーから履歴クリアやログイン時自動起動の切り替えができます

### 初回設定

1. アプリを起動します
2. 初回起動時に表示される案内に従って、macOS の「アクセシビリティ」を許可します
3. 必要に応じてアプリを再起動します

アクセシビリティ権限は、グローバルショートカットの検知と、自動ペーストの実行に必要です。

### 基本の使い方

#### 1. テキストを保存する

- どのアプリでも、保存したいテキストを選択します
- `⌘ + Shift + C` を押します
- 選択したテキストが Copit の履歴に保存されます

通常の `⌘ + C` とは別操作なので、Copit に保存するかどうかを自分で選べます。

#### 2. 履歴から貼り付ける

- 貼り付けたいアプリ上で `⌘ + Shift + V` を押します
- マウスカーソル付近に Copit の一覧が表示されます
- `Space` または `↑ / ↓` で項目を移動します
- `Enter` で選択中の項目を貼り付けます

#### 3. 一覧画面でできる操作

- `Enter`: 選択中の項目を貼り付け
- `Space`, `↑`, `↓`: 項目を移動
- `F`: お気に入りの切り替え
- `Delete` または `X`: 項目を削除
- `Esc`: 一覧を閉じる
- 項目をクリック: その項目をすぐ貼り付け
- 星ボタンをクリック: お気に入りの切り替え

### メニューバーからできること

メニューバーの Copit アイコンから、次の操作ができます。

- `ログイン時に自動起動`: macOS ログイン後に自動起動
- `履歴をクリア`: 保存済みの履歴を全削除
- `アクセシビリティ設定を開く...`: 権限設定画面を開く
- `終了`: アプリを終了

### 困ったとき

#### ショートカットが効かない

- Copit にアクセシビリティ権限が付与されているか確認してください
- 権限を付けた直後は、アプリの再起動が必要な場合があります

#### 履歴が出ない

- `⌘ + Shift + C` で一度テキストを保存してから `⌘ + Shift + V` を試してください
- 空白だけのテキストは履歴に追加されません

#### 項目が減っている

- 通常履歴は新しいものから最大 10 件まで保持されます
- 残しておきたい項目は `F` か星ボタンでお気に入りにしてください

## 開発者向け

### 概要

Copit は SwiftUI と AppKit で構成された macOS のメニューバー常駐アプリです。  
メインウィンドウは持たず、ステータスバー項目、グローバルショートカット、フローティングパネルで完結します。

### 主な仕様

- アプリ種別: `LSUIElement = true` のメニューバー常駐アプリ
- UI: SwiftUI ベースの履歴リスト
- イベント監視: `CGEventTap` によるグローバルショートカット処理
- 権限: アクセシビリティ権限が必須
- 自動起動: `ServiceManagement.SMAppService`
- 履歴保存: メモリ上のみ。現状は永続化なし

### ファイル構成

- `Copit/CopitApp.swift`: SwiftUI エントリポイント
- `Copit/AppDelegate.swift`: アプリ起動処理、メニューバー、権限確認、各コンポーネント接続
- `Copit/GlobalShortcutManager.swift`: `⌘⇧C` / `⌘⇧V` のグローバルショートカット処理
- `Copit/CopitPanelController.swift`: 貼り付け用パネルの表示、キー操作、疑似 `⌘V`
- `Copit/CopitListView.swift`: 履歴一覧 UI
- `Copit/CopitViewModel.swift`: 一覧 UI の表示状態
- `Copit/ClipboardManager.swift`: 履歴アイテム管理
- `Copit/ClipboardWatcher.swift`: システムクリップボード監視とコピー音制御
- `Copit/SoundManager.swift`: macOS 標準サウンド再生

### ローカルでの実行

1. Xcode で [Copit.xcodeproj](/Users/kondogenki/Copit/Copit.xcodeproj) を開きます
2. `Copit` ターゲットの macOS 実行先を選びます
3. `Run` で起動します
4. 初回起動後、アクセシビリティ権限を許可して再起動します

CLI から確認する場合:

```bash
xcodebuild -project Copit.xcodeproj -scheme Copit -configuration Debug build
```

### 実装メモ

- `⌘ + Shift + C` は内部的に一度通常のコピーを発行し、取得した文字列だけを履歴へ保存したうえで、元のクリップボード内容を復元します
- `⌘ + Shift + V` で開くパネルは、直前にアクティブだったアプリを覚えておき、項目確定後にそのアプリへ戻して `⌘V` を送っています
- 同じテキストを再保存した場合は重複追加せず、既存項目を先頭へ移動します
- お気に入り状態は、同一テキストの再保存時にも引き継がれます
