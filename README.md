# Copit

macOS メニューバーに常駐する、軽量クリップボード補助アプリ。  
⌘+Shift+C でテキストを専用履歴に保存し、⌘+Shift+V で一覧から素早く貼り付けできます。

## セットアップ

初回起動時にアクセシビリティ権限の許可ダイアログが表示されます。  
許可後、アプリを再起動してください。

## 操作方法

### グローバルショートカット

| キー | 操作 |
|------|------|
| ⌘ + Shift + C | 選択中のテキストを履歴に保存 |
| ⌘ + Shift + V | 履歴パネルを表示 |

### パネル内操作

| キー | 操作 |
|------|------|
| ↑ / ↓ / Space | 項目を移動 |
| Enter | 選択項目を貼り付けて閉じる |
| F | お気に入りトグル |
| Delete / X | 項目を削除 |
| Esc | 閉じる |

クリックで即貼り付け、星ボタンでお気に入りトグルも可能。  
通常履歴は最大 10 件。お気に入り登録した項目は上限なし。
